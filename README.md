## ✍️ O autorovi
Bývalý polda, který opustil svět paragrafů za vinylové drážky.  
Dnes nezaujatý pozorovatel života, co se tak šourá vedle děje a filosofuje nad cvrkotem světa.  
Čím je starší, tím méně ví – a je na to hrdý.
